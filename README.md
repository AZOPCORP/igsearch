# igsearch


### npm package to search inst * * * m without api 

Folowing my previous script to search inst * * * m without api I made igsearch.


```
npm install igsearch --save

```
usage: 
```javascript 

var ig = require('igsearch');

var search = 'glitch';

 ig.search(search,function(data){
   console.log(data);
 });


```
